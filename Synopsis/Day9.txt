DAY 9: MODERN JS RECAP & TYPESCRIPT INITIALIZATION
#############################################################################


1. ES6+ SYNTAX REFINEMENT
   Concept: Using modern shortcuts for cleaner, more readable code.
   - Default Parameters: Assigning fallback values to function arguments.
   - Spread & Rest Operators: Efficiently copying data and collecting arguments.
   - Destructuring: Position-based (arrays) and key-based (objects) extraction.

2. API INTEGRATION (AXIOS)
   Concept: Fetching real-world data using external libraries and async patterns.
   - Axios vs Fetch: Understanding the convenience of Axios (auto JSON parsing).
   - Async/Await: Managing asynchronous API calls in a readable, linear way.

3. TYPESCRIPT FOUNDATIONS
   - Initialization: Using 'npm init', 'npm install typescript', and 'npx tsc --init'.
   - Basic Types: Explicitly defining string, number, and boolean variables.
   - Type Annotation: Assigning types to function parameters and return values.

#############################################################################
KEY TAKEAWAYS (DAY 9)
#############################################################################
- Destructuring simplifies working with complex API response objects.
- TypeScript adds a layer of safety by catching type-related bugs during development.
- 'npx tsc' is the standard way to compile TypeScript into browser-readable JavaScript.
- Async/Await is preferred over .then() chains for better error handling and readability.
