JavaScript Object-Oriented Programming (OOP) Concepts 

1. Class and Object
A class is a blueprint for creating objects. Objects are instances of a class that contain properties and methods.

Example:
The Animal class contains a name property and a speak() method. Objects like dog and cat are created using this class.

--------------------------------------------

2. Encapsulation
Encapsulation means hiding data inside a class and controlling access using methods.

Example:
The BankAccount and Student classes use private variables (#balance and #marks). These variables cannot be accessed directly outside the class. Methods like deposit(), withdraw(), setMarks(), and getMarks() are used to access or modify the data safely.

Advantages:
- Protects data from direct modification
- Improves security
- Makes code more organized

--------------------------------------------

3. Abstraction
Abstraction means hiding internal implementation details and showing only important functionality to the user.

Example:
The CoffeeMachine and Car classes hide internal methods like #boilWater(), #grindBeans(), and #startEngine(). Users only interact with public methods like brew() and drive().

Advantages:
- Reduces complexity
- Makes code easier to use
- Improves maintainability

--------------------------------------------

4. Inheritance
Inheritance allows one class to acquire properties and methods from another class using the extends keyword.

Example:
The Admin class inherits from the User class.
The Dog class inherits from the Animal class.

Advantages:
- Code reusability
- Reduces duplicate code
- Creates parent-child relationships

--------------------------------------------

5. Polymorphism
Polymorphism means one method can perform different actions depending on the object.

Example:
The draw() method behaves differently in Circle and Square classes.
The sound() method behaves differently in Dog, Cat, and Cow classes.

Advantages:
- Flexibility
- Easy method overriding
- Better scalability

--------------------------------------------

Conclusion
Object-Oriented Programming in JavaScript helps developers write clean, reusable, secure, and organized code. The main OOP concepts are:
- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

These concepts improve code readability, maintenance, and reusability in real-world applications.
